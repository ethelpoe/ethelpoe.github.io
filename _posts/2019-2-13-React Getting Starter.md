You can use visual studio code with extensions: "prittier" and "simple react snippets"

paso 1: Instalar Node.js y npm

React requiere Node.js y npm (Node Package Manager). Si aún no los tienes instalados, sigue estos pasos:

1. **Descarga Node.js**: Ve a [nodejs.org](https://nodejs.org) y descarga la versión LTS (Long Term Support).
2. **Instala Node.js**: Sigue las instrucciones de instalación. Esto también instalará npm automáticamente.

paso 2:Crear un nuevo proyecto con Create React App

`npm create vite@latest react-project`

this command is when you already have a project

`npm install //in root folder`
`npm run dev`

to quit running the proyect
`ctrl  + c`

tip: your components name should be start with capital letter, otherwise react will try to find a default html element  

tip: you can write any javaScript code inside `{}`

tip: every component of react should just return one single element
