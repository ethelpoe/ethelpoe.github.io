## 1. Descripción General
This simple radio frequency (RF) based code lock incorporates a pair of 433MHz ready made RF transmitter and receiver modules. It is a remote-controlled, code-protected lock system. The code is fixed making it easy to implement. It requires very less power. You can remotely operate the solenoid lock from a distance of 30 metres or more depending on the RF modules used.
